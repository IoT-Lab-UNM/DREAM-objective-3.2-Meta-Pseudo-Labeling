# DREAM Objective 3.2: Meta-Pseudo-Labeling for Additive Manufacturing Segmentation

This repository contains public-facing artifacts for the DREAM Objective 3.2 study on synthetic layer-wise additive manufacturing segmentation and Meta-Pseudo-Labeling.

## Project Page

Interactive object inventory and documentation:

https://iot-lab-unm.github.io/DREAM-objective-3.2-Meta-Pseudo-Labeling/

## Repository Contents

- `docs/`: GitHub Pages website and object inventory artifacts.
- `scripts/`: selected reproducibility and reporting scripts.
- `configs/`: public configuration templates and sanitized experiment commands.
- `model_cards/`: draft Hugging Face dataset/model cards.
- `paper/`: static result summaries and selected figures.
- `data_examples/`: small example manifests or sample records.

## Dataset Summary

The study uses a synthetic layer-wise additive manufacturing segmentation dataset generated from 3D printable assets. The public inventory lists object-level metadata including serial ID, original object ID, layer count, and LVIS category metadata.

## Reproducibility Scope

This public repository provides selected scripts and artifacts sufficient to understand and reproduce the reported high-level workflow. Some internal research scripts, downstream experimental code, and full-scale data generation infrastructure are not included in this release.
